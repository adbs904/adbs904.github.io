Bootcamp 2020: Click The Targets

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, and randomness. I created a game based in which 8 randomly generated targets appear and need to be clicked.The timer stops once all targets have been destroyed so the aim is to complete the task as quickly as possible and try to improve your score.
